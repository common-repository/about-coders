Estimates the work of programmers who contribute to develop your Wordpress website plugins and themes.

== Description ==

This is a dashboard widget that uses AJAX to avoid load time excess. It displays the total and detailed estimations of bytes produced by each developers plugins and themes.

== Installation ==

1. Upload `this directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the dashboard
